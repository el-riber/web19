Remix Blog Project
Description
This project is a blog application built with Remix, designed to showcase dynamic content generation using Markdown files as the source for blog posts. It features a homepage listing all blog posts and individual pages for each post, leveraging Remix's nested routing capabilities for an organized and scalable structure.

Features
Dynamic content generation from Markdown files.
Homepage with a list of all blog posts.
Individual pages for each blog post.
(Optional) Nested routing for blog categories or tags.
Getting Started
Prerequisites
Before you begin, ensure you have the following installed on your system:

Node.js (version 12 or newer)
npm (comes with Node.js) or Yarn
Installation
Clone the repository
bash

git clone https://github.com/el-riber/web19.git your-remix-blog
cd your-remix-blog
Install dependencies
bash

npm install
or if you use Yarn:

bash

yarn
Running the Project Locally
Start the development server
bash

npm run dev
or with Yarn:

bash

yarn dev
Open your browser and navigate to http://localhost:57984/to view the project.
Building for Production
To build the project for production, run:

bash

npm run build
or with Yarn:

bash

yarn build
Followed by:

bash

npm start
or:

bash

yarn start
This will compile your Remix application for production and start it.
